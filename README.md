# 🚀 How to Run Locally

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/PrawinKumar514/weather-dashboard.git
```

## 2️⃣ Navigate to the Project Directory

```bash
cd weather-dashboard
```

## 3️⃣ Open the Project

Open the project folder in **Visual Studio Code**.

---

## 4️⃣ Get an OpenWeather API Key

1. Create an account at OpenWeather.
2. Generate a free API key.
3. Copy your API key.

---

## 5️⃣ Add Your API Key

Open:

```bash
js/script.js
```

Replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

With:

```javascript
const API_KEY = "YOUR_API_KEY_HERE";
```

---

## 6️⃣ Run the Project

Open:

```bash
index.html
```

Or use **Live Server** extension in VS Code.

# 📌 API Endpoint Used

```bash
https://api.openweathermap.org/data/2.5/weather
```

Example:

```bash
https://api.openweathermap.org/data/2.5/weather?q=Chennai&units=metric&appid=YOUR_API_KEY
```
